Historias de Usuario - Proyecto Lista de Compras

Historia 1: Crear lista de compras

Como persona encargada de hacer compras en mi hogar,
quiero poder crear una lista de compras desde la aplicación,
para organizar los productos que necesito antes de ir al supermercado.

Historia 2: Agregar productos

Como usuario de la aplicación,
quiero agregar productos a mi lista de compras,
para llevar un registro claro de lo que debo comprar.

Historia 3: Editar productos

Como usuario,
quiero editar la información de un producto de mi lista (nombre, cantidad, categoría),
para mantener mis datos actualizados cuando necesite modificar algo.

Historia 4: Marcar productos como comprados

Como comprador,
quiero marcar los productos que ya compré,
para saber cuáles elementos me faltan y evitar confusiones durante la compra.
